> **Signal Skills**
> AI 投研助手 · 数据查询 · 分析报告 · 多平台支持

将你的 AI 工具变成专业投研助手。Signal Skills 为 Cursor、Claude Code、Windsurf、GitHub Copilot 等支持 MCP 的平台提供金融数据查询与分析报告能力，内置投资理念与回复风格，在对话中直接完成投资分析。

## 不只是工具，是投研助手

Signal Skills 不只是数据接口的封装。技能包内置 AI Soul 人设系统，默认为**均衡型助手**——专业理性，结论先行，风控场景语气更硬。还提供毒舌老手、老练师父、冷静分析师等风格可选。

所有分析遵循「**价值为基、趋势为策、风控优先**」投资理念，单票仓位、行业集中度、回撤阈值都有明确标准，不说空话。

## 两层架构

技能包分为 **数据获取层** 和 **分析报告层**，共 14 个技能、18 个 MCP Tools：

```
用户提问
  ├── 查数据 → feed-data-tools（路由）→ 5 个数据子技能 → MCP tools → 返回数据
  └── 要分析 → analysis-router（路由）→ 7 个分析子技能
                                              ↓
                                   读取投资理念 + 调用 MCP tools 获取数据
                                              ↓
                                     按专业框架生成结构化报告
```

## AI Soul

技能包内置 **Soul** 人设系统，定义 AI 的身份、投资理念、回复风格与行为准则。默认为均衡型助手，也可切换为其他风格：

| 风格 | 特点 |
|------|------|
| **均衡型助手**（默认） | 专业理性，结论先行，风控时更直接 |
| 毒舌老手 | 犀利直接，擅长打断情绪化操作 |
| 老练师父 | 温和耐心，强调纪律与教学 |
| 冷静分析师 | 数据优先，弱情绪、强结构 |

切换方式：将对应风格的内容替换到 `soul.md` 即可。各风格完整内容见 [AI Soul](/signal-skills/souls)，支持一键复制。
