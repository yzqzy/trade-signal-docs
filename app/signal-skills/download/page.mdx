## 包内结构

```

│   └── check-update.ps1   # 版本检测与更新（Windows）
```

## 版本更新

技能包支持检测新版本并自动更新。

### 检测版本

在技能包目录下运行版本检测脚本：

```bash
# macOS / Linux
bash scripts/check-update.sh

# Windows PowerShell
.\scripts\check-update.ps1
```

### 自动更新

```bash
# macOS / Linux
bash scripts/check-update.sh --update
# 或
bash scripts/check-update.sh -u

# Windows PowerShell
.\scripts\check-update.ps1 -Update
```

### 更新机制

- **增量更新**：只同步新版本的文件，保留本地修改
- **强制保留**：`.mcp.json`（API Key）和 `soul.md`（自定义人设）不会被覆盖
- **更新内容**：`CLAUDE.md`、`.cursorrules`、`.windsurfrules`、技能目录等
