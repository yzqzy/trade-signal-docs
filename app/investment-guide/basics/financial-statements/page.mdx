## 为什么需要看财报？

财报是企业财务状况的"体检报告"，是评估<DocLink href="/investment-guide/core-concepts/enterprise-quality">企业质量</DocLink>的基础。学会快速看懂财报，可以帮助您：
- 了解企业的真实财务状况
- 判断企业是否值得投资
- 识别财务风险信号

根据<DocLink href="/investment-guide/investment-philosophy">投资理念</DocLink>，财报阅读应该关注**财务稳健性**、**盈利质量**和**风险控制**。

## 三大财务报表

### 1. 资产负债表（Balance Sheet）

**作用**：反映企业在某个时点的财务状况，包括资产、负债和所有者权益。

**核心公式**：
```
资产 = 负债 + 所有者权益
```

**重点关注**：
- **总资产**：企业规模
- **总负债**：企业债务水平
- **所有者权益**：股东权益
- **资产负债率**：总负债 ÷ 总资产（目标 < 60%）

**如何快速判断**：
- 查看**资产负债率**：< 60% 为健康，> 60% 需警惕
- 查看**流动比率**：流动资产 ÷ 流动负债（> 1 为健康）
- 查看**所有者权益**：是否持续增长

### 2. 利润表（Income Statement）

**作用**：反映企业在一段时间内的经营成果，包括收入、成本和利润。

**核心公式**：
```
净利润 = 营业收入 - 营业成本 - 期间费用 - 所得税
```

**重点关注**：
- **营业收入**：企业收入规模
- **营业成本**：产品成本
- **毛利率**：(营业收入 - 营业成本) ÷ 营业收入
- **净利润**：最终盈利
- **ROE**：净利润 ÷ 所有者权益（目标 ≥ 15%）

**如何快速判断**：
- 查看**营业收入**：是否持续增长
- 查看**毛利率**：是否稳定或改善
- 查看**净利润**：是否稳定增长
- 查看**ROE**：是否 ≥ 15%

### 3. 现金流量表（Cash Flow Statement）

**作用**：反映企业在一段时间内的现金流入和流出情况。

**三大现金流**：
- **经营现金流**：经营活动产生的现金流（最重要）
- **投资现金流**：投资活动产生的现金流
- **筹资现金流**：筹资活动产生的现金流

**重点关注**：
- **经营现金流**：是否 > 0，是否 > 净利润
- **自由现金流**：经营现金流 - 资本支出（目标 > 0）
- **每股经营现金流**：经营现金流 ÷ 总股本

**如何快速判断**：
- 查看**经营现金流**：> 0 为健康，> 净利润 为优秀
- 查看**自由现金流**：> 0 为健康，持续改善为优秀
- 查看**现金流/净利润**：> 0.8 说明盈利质量高

## 快速判断企业质量

根据投资理念，快速判断企业质量需要关注以下核心指标：

### 第一步：财务稳健性（30%权重）

**核心指标**：
- **自由现金流**：> 0（优先持续改善，近3年）
- **ROE**：≥ 12%（优先 ≥ 15%，且维持稳定或上升趋势）
- **资产负债率**：< 60%，且呈下降趋势
- **每股经营现金流**：> 0 且优于行业平均

**快速判断**：
- ✅ **优秀**：自由现金流 > 0 + ROE ≥ 15% + 负债率 < 60%
- ⚠️ **一般**：部分指标不达标
- ❌ **较差**：自由现金流 < 0 或 ROE < 12% 或 负债率 > 60%

### 第二步：盈利质量（25%权重）

**核心指标**：
- **盈利稳定性**：近5年盈利稳定，无连续两年亏损
- **净利润含金量**：经营现金流/净利润 > 0.8
- **毛利率、净利率**：维持稳定或改善

**快速判断**：
- ✅ **优秀**：盈利稳定 + 现金流/净利润 > 0.8 + 毛利率稳定
- ⚠️ **一般**：盈利波动或现金流/净利润 < 0.8
- ❌ **较差**：连续亏损或现金流/净利润 < 0.5

### 第三步：成长性（10%权重）

**核心指标**：
- **营业收入增长率**：是否持续增长
- **净利润增长率**：是否稳定增长
- **3年复合增长率**：是否 > 10%

**快速判断**：
- ✅ **优秀**：3年复合增长率 > 15%
- ⚠️ **一般**：3年复合增长率 5-15%
- ❌ **较差**：增长率 < 5% 或负增长

## 财报阅读流程

### 第一步：看利润表（了解盈利能力）

1. 查看**营业收入**：是否持续增长
2. 查看**毛利率**：是否稳定或改善
3. 查看**净利润**：是否稳定增长
4. 查看**ROE**：是否 ≥ 15%

**快速判断**：如果ROE < 12%或盈利不稳定，可以跳过，不需要继续看。

### 第二步：看现金流量表（了解现金流质量）

1. 查看**经营现金流**：是否 > 0
2. 查看**自由现金流**：是否 > 0
3. 查看**现金流/净利润**：是否 > 0.8

**快速判断**：如果经营现金流 < 0 或现金流/净利润 < 0.5，需要警惕。

### 第三步：看资产负债表（了解财务风险）

1. 查看**资产负债率**：是否 < 60%
2. 查看**流动比率**：是否 > 1
3. 查看**所有者权益**：是否持续增长

**快速判断**：如果负债率 > 60% 且呈上升趋势，需要特别警惕。

### 第四步：综合判断

根据以上三步的结果，综合判断企业质量：
- ✅ **优秀**：所有指标都达标，可以重点关注
- ⚠️ **一般**：部分指标不达标，需要进一步评估
- ❌ **较差**：多个指标不达标，建议规避

## 重点关注指标速查表

| 指标 | 优秀标准 | 一般标准 | 较差标准 | 重要性 |
|------|---------|---------|---------|--------|
| **ROE** | ≥ 15% | 12-15% | < 12% | ⭐⭐⭐⭐⭐ |
| **自由现金流** | > 0 且持续改善 | > 0 | < 0 | ⭐⭐⭐⭐⭐ |
| **资产负债率** | < 60% 且下降 | < 60% | > 60% | ⭐⭐⭐⭐⭐ |
| **经营现金流** | > 净利润 | > 0 | < 0 | ⭐⭐⭐⭐ |
| **毛利率** | 稳定或上升 | 稳定 | 下降 | ⭐⭐⭐⭐ |
| **净利润增长率** | > 15% | 5-15% | < 5% | ⭐⭐⭐ |
| **现金流/净利润** | > 0.8 | 0.5-0.8 | < 0.5 | ⭐⭐⭐ |

## 常见财务风险信号

### ⚠️ 需要警惕的信号

1. **自由现金流持续为负**：企业可能现金流紧张
2. **资产负债率快速上升**：财务风险增加
3. **经营现金流 < 净利润**：盈利质量可能有问题
4. **ROE持续下降**：盈利能力下降
5. **连续两年亏损**：企业可能经营困难

### ❌ 应该规避的信号

1. **自由现金流长期为负**：企业可能无法持续经营
2. **资产负债率 > 70%**：财务风险很高
3. **经营现金流长期为负**：企业可能无法产生现金
4. **ROE < 10%**：盈利能力较差
5. **连续三年亏损**：企业可能面临退市风险

## 常见误区

### ❌ 误区1：只看净利润

**错误**：只看净利润高就认为企业好
**正确**：需要综合看ROE、现金流、负债率等多个指标

### ❌ 误区2：忽视现金流

**错误**：只看利润，不看现金流
**正确**：现金流比利润更重要，自由现金流 > 0 是关键

### ❌ 误区3：只看单一指标

**错误**：只看ROE高就买入
**正确**：需要综合评估财务稳健性、盈利质量、成长性等多个维度

### ❌ 误区4：忽视趋势

**错误**：只看当前数据，不看历史趋势
**正确**：需要关注指标的变化趋势，持续改善比单点数据更重要
