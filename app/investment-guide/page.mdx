## 为什么需要学习投资知识？

投资是让财富增值的重要途径，但投资有风险，需要系统化的知识和正确的理念。本投资指南旨在帮助您：

- 📚 **建立正确的投资理念** - 理解投资本质，避免常见误区
- 🎯 **掌握核心概念** - 深入理解安全边际、趋势共振等核心概念
- 💡 **学习基础知识** - 从股票、ETF到投资策略，循序渐进
- 🛡️ **提升投资技能** - 学会分析、筛选和决策

## 学习路径建议

建议先通读 [投资理念](/investment-guide/investment-philosophy) 与 [新手指南](/investment-guide/beginner-guide/how-to-start)，再使用工具，形成「理念 → 心法 → 概念 → 工具」的闭环。

### 新手路径
1. [新手指南](/investment-guide/beginner-guide/how-to-start) - 了解如何开始投资，建立正确的投资观念
2. [投资理念](/investment-guide/investment-philosophy) - 理解顶层理念，建立投资框架
3. [投资心法](/investment-guide/investment-mindset/stop-loss-and-risk) - 建立止损与规则化执行意识
4. [基础知识](/investment-guide/basics/glossary) - 学习投资术语、股票和ETF基础
5. [核心概念](/investment-guide/core-concepts/safety-margin) - 深入理解安全边际、趋势共振等核心概念
6. **用工具实践**：学完核心概念后 → [股票筛选器](/signal-client/guides/stock-screener)、[ETF 筛选器](/signal-client/guides/etf-screener) 应用筛选与估值分位

掌握核心概念后，可进一步学习 [市场分析](/investment-guide/market-analysis/market-indicators)，理解看板与宏观指标在决策中的用法。

### 进阶路径
1. 复习 [投资理念](/investment-guide/investment-philosophy) 与 [核心概念](/investment-guide/core-concepts)
2. 学习 [市场分析](/investment-guide/market-analysis/market-indicators) - 掌握宏观分析、行业分析等进阶方法（参见上方「投资指南 ↔ 工具功能」对照表）
3. **用工具实践**：学完市场分析后 → [看板](/signal-client/guides/dashboard)、[热图](/signal-client/guides/hotmap)、[宏观指标](/signal-client/guides/macro-indicators) 做趋势与复盘
4. 学习 [工具方法论](/signal-client/methodology)（K 线图、指标、预设条件），结合 [计划管理](/signal-client/guides/trading-plans) 做系统化复盘

详细功能说明与完整工作流见 [投资分析工具](/signal-client)。
