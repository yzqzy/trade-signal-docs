## v0.1.13 - 2026-01-13

### 新增

- **AI 技能框架重构**：重构 AI 分析模块，引入 Skills 架构和 LangChain 集成
  - 技能化架构：将不同分析能力拆分为独立技能模块（Helper Skills、Data Skills、Analysis Skills）
  - 智能技能调度：根据问题类型自动选择和执行相应技能
  - 技能组合：支持多个技能协同工作，提供更全面的分析
  - 依赖管理：自动解析技能依赖关系，确保执行顺序
- **股票池分页功能**：股票池数据支持分页显示
  - 分页显示：支持手动分页，每页显示 50 条记录
  - 性能优化：仅加载当前页的股票数据，提升大数据量场景下的加载速度
  - 分页信息：显示总记录数和当前页信息

## v0.1.12 - 2026-01-11

### 新增

- **股票池管理**：股票池功能，支持批量导入和计划创建
  - 批量导入：支持从筛选结果批量导入到股票池
  - 计划创建：从股票池直接创建交易计划
- **自定义筛选条件**：ETF 和股票筛选器支持保存和管理自定义筛选条件
  - 保存筛选条件：将常用筛选条件保存为自定义预设
  - 管理自定义预设：编辑、删除已保存的筛选条件
  - 快速应用：一键应用已保存的筛选条件
- **数据导出功能**：支持将筛选结果导出为 CSV 和 Excel 格式
  - CSV 导出：标准 CSV 格式，兼容 Excel
  - Excel 导出：完整格式支持，包括 ETF 类型代码和布尔字段处理
  - 支持选股和 ETF 筛选结果导出
- **在线申请授权密钥**：授权注册页面新增在线申请授权密钥按钮

### 优化

- 优化 AI 分析流式消息显示布局
- 优化股票数据处理和 API 集成

## v0.1.10 - 2026-01-06

### 新增

- **AI 股票分析模块**：集成大模型 API，基于现有数据生成深度分析报告
  - 支持多提供商：DeepSeek、通义千问、GLM、Moonshot、自定义 API
  - 流式响应：实时显示分析结果
  - 智能股票识别：从问题中自动提取股票代码
  - 意图分析：自动识别问题意图和数据需求
  - 数据源匹配：智能匹配所需数据源并聚合
- **对话管理**：历史记录查看和管理
  - 会话列表、对话详情、多轮对话支持
  - 会话标题自动生成、时间戳显示
  - 数据快照查看：查看分析使用的完整数据
- **模板管理**：提示词模板创建、编辑、删除
  - 模板导入/导出功能
  - AI 优化模板：使用 AI 优化提示词模板
  - 模板选择：输入 @ 快速选择模板
  - 数据源绑定：模板自动关联所需数据源
- **配置管理**：API 配置和场景化模型选择
  - 支持多场景配置（意图分析、数据需求、股票分析等）
  - 智能模型选择：根据场景自动选择最合适的模型
  - 数据缓存：双层缓存机制，提升响应速度
- **用户体验优化**：数据源指南、进度提示、取消功能、复制功能
