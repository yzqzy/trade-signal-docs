> **专业的投资分析生产力工具**  
> 盘后复盘 · 热点追踪 · 深度分析 · 智能筛选

## 产品定位

这是一款专为 A 股投资者设计的**桌面生产力工具**，帮助您高效完成盘后复盘、热点追踪和投资决策。告别繁琐的数据查找和零散的信息收集，在一个工具中完成从市场分析到个股研究的全流程工作。

## 核心使用场景

### 📊 盘后复盘

**每日收盘后的标准工作流：**

1. **市场总览** - 快速了解当日市场整体表现
   - 涨跌家数统计、市场温度指标
   - 全球市场对比，把握整体环境

2. **资金流向分析** - 识别资金动向
   - 资金流入流出统计
   - 行业资金流向排名
   - 资金流向趋势图表

3. **市场主线识别** - 发现当前市场热点
   - 自动识别市场主线
   - 板块轮动趋势分析
   - 热点题材追踪

4. **热力图分析** - 直观把握市场结构
   - 行业/概念板块热力图
   - 多维度指标可视化（涨跌幅、成交额、市值等）
   - 交互式树状图展示

### 🔥 热点关注

**持续追踪市场热点和投资机会：**

- **题材库管理** - 建立自己的题材概念库，追踪题材热度变化
- **市场要闻** - 重要资讯汇总，不错过关键信息
- **个股深度分析** - 对热点个股进行全方位研究
- **股东变动追踪** - 关注机构和大股东动向

### 🔍 智能筛选

**基于投资理念的筛选工具：**

- **多维度选股** - 安全边际、趋势共振、组合稳健等指标组合筛选
- **ETF 筛选器** - 估值分位分析，找到合适的 ETF 产品
- **预设方案** - 内置多种筛选策略，快速定位目标标的

## 核心功能

- 📊 **看板总览** - 市场总览、资金流向、市场主线，一盘掌握
- 🔥 **热力图** - 行业/概念板块可视化，直观发现热点
- 🔍 **智能筛选** - 多维度选股与 ETF 筛选，精准定位
- 📈 **深度分析** - 个股全方位分析，安全边际与趋势共振评估
- 📰 **资讯服务** - 市场要闻汇总，重要信息不错过
- 🎯 **题材库** - 题材概念管理，追踪热点轮动

## 典型工作流示例

### 盘后复盘流程（15:00 - 16:00）

1. **打开看板** → 查看市场总览，了解当日整体表现
2. **分析资金流向** → 查看行业资金流入流出排名，识别资金动向
3. **识别市场主线** → 查看市场主线分析，发现当前热点
4. **查看热力图** → 通过热力图直观了解板块表现
5. **筛选关注标的** → 使用选股工具，基于热点和资金流向筛选个股
6. **深度分析** → 对筛选出的个股进行深度分析，评估安全边际和趋势
7. **查看要闻** → 浏览市场要闻，了解重要信息

### 热点追踪流程

1. **题材库管理** → 建立和更新关注的题材概念
2. **市场主线监控** → 每日查看市场主线变化
3. **热力图观察** → 通过热力图发现异动板块
4. **个股分析** → 对热点个股进行快速分析
5. **股东追踪** → 关注机构和大股东动向
