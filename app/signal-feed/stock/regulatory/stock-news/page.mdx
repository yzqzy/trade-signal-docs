# 个股新闻

个股新闻。

**HTTP 方法与路径**：`GET /api/v1/stock/news`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `code` | string | 是 | 股票代码 |
| `pageIndex` | number | 否 |  |
| `pageSize` | number | 否 |  |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。

### 响应示例

对应请求 `GET /api/v1/stock/news?code=000001`。字段以线上为准。

```json
{
  "success": true,
  "data": {
    "total": 1,
    "list": []
  }
}
```

## 请求示例

```bash
curl -sS -H "Authorization: Bearer <API_KEY>" "http://localhost:4000/api/v1/stock/news?code=000001"
```
