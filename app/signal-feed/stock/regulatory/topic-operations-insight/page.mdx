# 运营与管理洞察聚合

运营与管理洞察聚合。

**HTTP 方法与路径**：`GET /api/v1/stock/topic/operations-insight/{code}`

## 入参

### Path

| 参数 | 位置 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| `code` | path | string | 是 | 股票代码 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `year` | string | 否 |  |
| `reportDate` | string | 否 | `YYYYMM` 或 `YYYY-MM-DD` |
| `timeRange` | string | 否 |  |
| `governanceLimit` | number | 否 |  |
| `forecastPageSize` | number | 否 |  |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
