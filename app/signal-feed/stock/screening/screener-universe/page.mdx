# 选股 Universe（Screener）

选股 Universe（Screener）。

**HTTP 方法与路径**：`GET /api/v1/stock/screener/universe`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `market` | 'CN_A'|'HK' | 否 | 默认 CN_A |
| `page` | number | 否 | 默认 1 |
| `pageSize` | number | 否 | 默认 100，最大 500 |
| `filter` | string | 否 | 附加筛选 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
