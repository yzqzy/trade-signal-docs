# 同业可比池（自动）

同业可比池（自动）。

**HTTP 方法与路径**：`GET /api/v1/stock/peers/{code}`

## 入参

### Path

| 参数 | 位置 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| `code` | path | string | 是 | 股票代码 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `year` | string | 否 |  |
| `topN` | number | 否 | 1–20 |
| `sortColumn` | string | 否 |  |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
