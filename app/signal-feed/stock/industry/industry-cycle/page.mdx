# 行业周期快照

行业周期快照。

**HTTP 方法与路径**：`GET /api/v1/stock/industry/cycle/{code}`

## 入参

### Path

| 参数 | 位置 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| `code` | path | string | 是 | 股票代码 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `year` | string | 否 |  |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。

### 响应示例

对应请求 `GET /api/v1/stock/industry/cycle/000001`。字段以线上为准。

```json
{
  "success": true,
  "data": {
    "code": "000001",
    "industryName": "股份制银行Ⅲ",
    "year": "2025",
    "previousYear": "2024",
    "cyclicality": "weak",
    "position": "middle",
    "confidence": "medium",
    "metrics": {
      "sampleSizeCurrent": 9,
      "sampleSizePrevious": 9,
      "revenueAllYearYoY": -0.016836441477779008,
      "parentNiAllYearYoY": -0.0007048848721322868
    }
  }
}
```

## 请求示例

```bash
curl -sS -H "Authorization: Bearer <API_KEY>" "http://localhost:4000/api/v1/stock/industry/cycle/000001"
```
