# 财务画像

财务画像。

**HTTP 方法与路径**：`GET /api/v1/stock/diagnosis/financial/portrait/{code}`

## 入参

### Path

| 参数 | 位置 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| `code` | path | string | 是 | 股票代码 |


### Query

| Query 参数 | 无 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。

### 响应示例

对应请求 `GET /api/v1/stock/diagnosis/financial/portrait/000001`。字段以线上为准。

```json
{
  "success": false,
  "data": null,
  "message": "Failed to get financial portrait"
}
```

## 请求示例

```bash
curl -sS -H "Authorization: Bearer <API_KEY>" "http://localhost:4000/api/v1/stock/diagnosis/financial/portrait/000001"
```
