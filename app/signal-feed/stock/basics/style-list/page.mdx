# 风格列表

风格列表。

**HTTP 方法与路径**：`GET /api/v1/stock/style/list`

## 入参

### Path

| 路径参数 | 无 |


### Query

| Query 参数 | 无 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。

### 响应示例

对应请求 `GET /api/v1/stock/style/list`。字段以线上为准。

```json
{
  "success": true,
  "data": {
    "list": [
      {
        "code": "2025三季报预增",
        "name": "2025三季报预增"
      },
      "..."
    ]
  }
}
```

## 请求示例

```bash
curl -sS -H "Authorization: Bearer <API_KEY>" "http://localhost:4000/api/v1/stock/style/list"
```
