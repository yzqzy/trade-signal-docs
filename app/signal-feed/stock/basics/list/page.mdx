# 股票批量查询

股票批量查询。

**HTTP 方法与路径**：`GET /api/v1/stock/list`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `codes` | string | 是 | 逗号分隔，6 位或带后缀 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
