# 年报财务历史（合并）

年报财务历史（合并）。

**HTTP 方法与路径**：`GET /api/v1/stock/financial/history/{code}`

## 入参

### Path

| 参数 | 位置 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| `code` | path | string | 是 | 股票代码 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `reportType` | string | 否 | 报表类型，可选 |
| `years` | number | 否 | 年数 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
