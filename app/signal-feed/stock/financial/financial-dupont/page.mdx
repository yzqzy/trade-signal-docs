# 年报杜邦分析序列

年报杜邦分析序列。

**HTTP 方法与路径**：`GET /api/v1/stock/financial/dupont/{code}`

## 入参

### Path

| 参数 | 位置 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| `code` | path | string | 是 | 股票代码 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `reportType` | string | 否 |  |
| `years` | number | 否 |  |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
