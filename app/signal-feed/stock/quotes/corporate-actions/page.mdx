# 企业行动

企业行动。

**HTTP 方法与路径**：`GET /api/v1/stock/corporate-actions`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `code` | string | 是 | 股票代码 |
| `from` | string | 否 | 除权除息日过滤起点 |
| `to` | string | 否 | 终点 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。

### 响应示例

对应请求 `GET /api/v1/stock/corporate-actions?code=000001`。字段以线上为准。

```json
{
  "success": true,
  "data": [
    {
      "code": "000001",
      "actionType": "dividend",
      "exDate": "2025-10-15",
      "recordDate": "2025-10-14",
      "cashDividendPerShare": 0.236
    },
    "..."
  ]
}
```

## 请求示例

```bash
curl -sS -H "Authorization: Bearer <API_KEY>" "http://localhost:4000/api/v1/stock/corporate-actions?code=000001"
```
