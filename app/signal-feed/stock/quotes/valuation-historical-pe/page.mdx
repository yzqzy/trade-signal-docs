# 历史 PE TTM

历史 PE TTM。

**HTTP 方法与路径**：`GET /api/v1/stock/valuation/historical-pe`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `code` | string | 是 | 股票代码 |
| `interval` | string | 否 | 默认 `60` |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
