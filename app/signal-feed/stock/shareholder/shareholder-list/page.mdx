# 股东持仓列表

股东持仓列表。

**HTTP 方法与路径**：`GET /api/v1/stock/shareholder/list`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | string | 否 | `top10`/`top10Float` |
| `code` | string | 否 | **股东代码**（勿传 6 位股票代码） |
| `endDate` | string | 否 | `YYYY-MM-DD` |
| `page` | number | 否 |  |
| `pageSize` | number | 否 |  |
| `holdNumChange` | string | 否 | 持股变动类型 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
