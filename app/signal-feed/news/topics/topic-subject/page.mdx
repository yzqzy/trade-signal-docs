# 专题资讯流

按 `subjectKey` 或 `subjectId` 拉取专题资讯流，支持时间游标分页。

**HTTP 方法与路径**：`GET /api/v1/news/topic-subject`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `subjectKey` | string | 否* | 与 `subjectId` 二选一；取值示例：`morning_brief`、`news_broadcast` 等 |
| `subjectId` | number | 否* | 专题数字 ID |
| `pageSize` | number | 否 | 默认 20 |
| `lastTime` | number | 否 | 秒级时间戳游标 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。

## 请求示例

```bash
curl -sS -H "Authorization: Bearer <API_KEY>" "http://localhost:4000/api/v1/news/topic-subject?subjectKey=morning_brief&pageSize=5"
```


> *`subjectKey` 与 `subjectId` 至少填一个。
