# 快讯要闻

快讯与要闻流，游标分页。

**HTTP 方法与路径**：`GET /api/v1/news/feed`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `cursor` | string | 否 | 下一页游标，首次不传 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
