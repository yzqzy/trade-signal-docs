# 要闻摘要

新闻联播要闻、精选等摘要。

**HTTP 方法与路径**：`GET /api/v1/news/headlines`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `page` | number | 否 | 默认 0 |
| `pageSize` | number | 否 | 默认 10 |
| `keyword` | string | 否 | 默认「新闻联播要闻」 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
