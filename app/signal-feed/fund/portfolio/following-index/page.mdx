# 跟踪指数

跟踪指数信息。

**HTTP 方法与路径**：`GET /api/v1/fund/following-index/{code}`

## 入参

### Path

| 参数 | 位置 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| `code` | path | string | 是 | 基金代码 |


### Query

| Query 参数 | 无 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。

### 响应示例

对应请求 `GET /api/v1/fund/following-index/159915`。字段以线上为准。

```json
{
  "success": true,
  "data": {
    "securityCode": "399006.SZ",
    "securityName": "创业板指",
    "securityType": "01",
    "price": 3667.79,
    "changePercent": -1.4101,
    "securityCount": 100,
    "change": -52.46,
    "yearlyPerformance": 89.47,
    "listingTimeFlag": "02"
  }
}
```

## 请求示例

```bash
curl -sS -H "Authorization: Bearer <API_KEY>" "http://localhost:4000/api/v1/fund/following-index/159915"
```
