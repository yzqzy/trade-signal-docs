# 市场热力图

全市场热力图节点；可传上次返回的 hash 做增量。

**HTTP 方法与路径**：`GET /api/v1/market/hotmap`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `quotedata_hash` | string | 否 | 上次响应中的增量 hash |

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
