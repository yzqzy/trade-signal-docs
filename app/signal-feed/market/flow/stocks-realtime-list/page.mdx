# 股票实时列表

个股或标的实时列表（非板块资金流）。

**HTTP 方法与路径**：`GET /api/v1/market/stocks/realtime-list`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `market` | string | 是 | `cn_a` 沪京深合并A股 · `bj` 北证 · `cyb` 创业板 · `kcb` 科创板 · `hk` / `hk_main` / `hk_gem` 港股等（完整取值以线上 OpenAPI `/docs` 为准） |
| `page` | number | 否 | 默认 1 |
| `pageSize` | number | 否 | 默认 50 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。
