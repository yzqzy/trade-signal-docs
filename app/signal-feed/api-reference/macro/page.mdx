## 宏观接口

Signal Feed 提供完整的宏观经济数据接口，支持PMI、CPI、GDP、货币供应量等宏观经济指标。

## 基础信息

- **Base URL**: `http://localhost:4000/api/v1`
- **认证方式**: Bearer Token
- **数据格式**: JSON
- **接口前缀**: `/macro`

## 主要接口

### 经济指标

#### PMI 数据
```http
GET /api/v1/macro/pmi?page=1&pageSize=20
```
分页获取采购经理指数数据。

#### PMI 系列
```http
GET /api/v1/macro/pmi/series
```
获取 PMI 系列数据（图表用）。

#### CPI 数据
```http
GET /api/v1/macro/cpi?page=1&pageSize=20
```
获取居民消费价格指数数据。

#### CPI 系列
```http
GET /api/v1/macro/cpi/series
```
获取 CPI 系列数据。

#### PPI 数据
```http
GET /api/v1/macro/ppi?page=1&pageSize=20
```
获取工业品出厂价格指数数据。

#### PPI 系列
```http
GET /api/v1/macro/ppi/series
```
获取 PPI 系列数据。

#### GDP 数据
```http
GET /api/v1/macro/gdp?page=1&pageSize=20
```
获取国内生产总值数据。

#### GDP 系列
```http
GET /api/v1/macro/gdp/series
```

获取 GDP 系列数据。

## 使用示例

```javascript
const API_TOKEN = 'your_api_token';
const BASE_URL = 'http://localhost:4000/api/v1';

// 获取PMI数据
async function getPMI(page = 1, pageSize = 20) {
  const params = new URLSearchParams({
    page: page.toString(),
    pageSize: pageSize.toString()
  });
  const response = await fetch(`${BASE_URL}/macro/pmi?${params}`, {
    headers: {
      'Authorization': `Bearer ${API_TOKEN}`
    }
  });
  return response.json();
}

// 获取PMI系列数据（图表用）
async function getPMISeries() {
  const response = await fetch(`${BASE_URL}/macro/pmi/series`, {
    headers: {
      'Authorization': `Bearer ${API_TOKEN}`
    }
  });
  return response.json();
}

// 获取CPI系列数据
async function getCPISeries() {
  const response = await fetch(`${BASE_URL}/macro/cpi/series`, {
    headers: {
      'Authorization': `Bearer ${API_TOKEN}`
    }
  });
  return response.json();
}

// 获取GDP系列数据
async function getGDPSeries() {
  const response = await fetch(`${BASE_URL}/macro/gdp/series`, {
    headers: {
      'Authorization': `Bearer ${API_TOKEN}`
    }
  });
  return response.json();
}
```
