# 社会消费品零售总额分页

分页返回该指标表格数据。

**HTTP 方法与路径**：`GET /api/v1/macro/total-retail`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `page` | undefined | undefined | undefined |
| `pageSize` | undefined | undefined | undefined |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。

### 响应示例

对应请求 `GET /api/v1/macro/total-retail?page=1&pageSize=20`。字段以线上为准。

```json
{
  "success": true,
  "data": {
    "pages": 11,
    "data": [
      {
        "REPORT_DATE": "2026-03-01 00:00:00",
        "TIME": "2026年03月份",

        "RETAIL_TOTAL_SAME": 1.7,
        "RETAIL_TOTAL_SEQUENTIAL": null,
        "RETAIL_TOTAL_ACCUMULATE": 127695,
        "RETAIL_ACCUMULATE_SAME": 2.4

  }
}
```

## 请求示例

```bash
curl -sS -H "Authorization: Bearer <API_KEY>" "http://localhost:4000/api/v1/macro/total-retail?page=1&pageSize=20"
```
