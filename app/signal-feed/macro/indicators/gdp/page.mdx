# GDP 分页

分页返回该指标表格数据。

**HTTP 方法与路径**：`GET /api/v1/macro/gdp`

## 入参

### Path

| 路径参数 | 无 |


### Query

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `page` | undefined | undefined | undefined |
| `pageSize` | undefined | undefined | undefined |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。

## 请求示例

```bash
curl -sS -H "Authorization: Bearer <API_KEY>" "http://localhost:4000/api/v1/macro/gdp?page=1&pageSize=20"
```
