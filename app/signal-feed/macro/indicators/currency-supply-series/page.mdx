# 货币供应量序列

返回该指标的时序序列数据（图表用）。

**HTTP 方法与路径**：`GET /api/v1/macro/currency-supply/series`

## 入参

### Path

| 路径参数 | 无 |


### Query

| Query 参数 | 无 |


## 响应

> **响应外壳**：成功时 JSON 含 `success`、`data`、可选 `message`，见 [统一响应体](/signal-feed/api-reference#api-response-envelope)。

### 响应示例

对应请求 `GET /api/v1/macro/currency-supply/series`。字段以线上为准。

```json
{
  "success": true,
  "data": {
    "pages": 1,
    "data": [
      {
        "REPORT_DATE": "2026-03-01 00:00:00",
        "BASIC_CURRENCY_SEQUENTIAL": 1.33088464,
        "CURRENCY_SEQUENTIAL": 2.92809245,
        "FREE_CASH_SEQUENTIAL": -2.87487005
      },
      "..."
    ],
    "count": 219
  }
}
```

## 请求示例

```bash
curl -sS -H "Authorization: Bearer <API_KEY>" "http://localhost:4000/api/v1/macro/currency-supply/series"
```
